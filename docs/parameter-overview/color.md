**Parameter:** `color`

**Default:** `0,0,0,100`

The `color` parameter sets the color of the QR code in r,g,b,a format. Example: "0,0,0,100".

### Example

GET
```http
/api/qr?data=sample-data&color=50,200,100,100
```
GET
```http
/api/qr?data=sample-data&color=200,20,150,100
```
